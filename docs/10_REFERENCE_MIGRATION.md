# DOC-10: Reference Migration (Archived)

> **Status**: Archived. 이 문서는 Python/Textual 기반 구현 시절의 exporter/grafana 참조 데이터 마이그레이션 기록이며, Java/Lanterna 기반으로 재작성된 현재 DIT에는 해당하지 않는다.

## 이력

- Python/Textual 기반 TermDB Monitor 시절에 `exporter/`, `grafana/` 디렉터리의 참조 데이터를 프로젝트 내부(`src/dit_monitor/`)로 흡수한 작업 기록
- 현재 DIT는 Java 단일 JAR로 재작성되어 해당 구조가 더 이상 존재하지 않음
- `references/` 디렉터리는 `.gitignore`에 등록하여 참고용으로만 유지
